# Seed Orchard UI

A static front-end concept where you enter a few simple inputs, plant a seed, and harvest actionable output cards.

## Run
- Open `index.html` directly, or
- Serve locally:
  - `python -m http.server 8080`
  - open `http://localhost:8080/seed_orchard_ui/`

## Inputs
- Seed Name
- Core Goal
- Audience
- Harvest Size
- Intensity

## Output
- Animated growth phases
- 3 to 5 generated "fruit" cards
- Each fruit includes:
  - output concept
  - reason it matters
  - first action
  - effort + signal tags
