Quick Start (Local)

Run from the repo root. If you are serving pages locally, prefer python -m http.server 8080 to avoid file:// fetch/audio issues. 

cd full_seed_starter/av_seedgen_python_profiles
python seedgen.py --list-profiles

# Example (see README for full flags)
python seedgen.py --profile onboarding --name founder_welcome --title "Founder Welcome" --tagline "Earn your badge." --zip

Outputs

The goal is a runnable pack artifact (ZIP) that follows the template contract: it should load data/bytecast_ep_profile.json without edits, and keep links relative for GitHub Pages.
