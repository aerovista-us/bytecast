# QA Checklists (Internal)

## Web (LaunchPad / Quest Page / Mini-Game)
- Mobile layout passes
- Performance acceptable
- Contact flow tested
- Analytics (if included) verified
- Domain + SSL verified
- CTA visible above the fold
- Basic SEO fields present
- Accessibility quick pass (contrast, labels, focus)

## Audio (Sonic Logo / Podcast Pack / Track Polish)
- Loudness target consistent
- No clipping
- Stems exported (if included)
- File naming standard
- Client preview link works

## Design (Brand Kit / Social Pack / Print)
- Export sizes correct
- Fonts licensed/converted properly
- Print bleed/safe zones checked (if applicable)
- Source files packaged

## Drone (Weather dependent)
- Flight plan + safety checks done
- Shots match brief angles
- Edits consistent and labeled
- Deliverables organized for client
