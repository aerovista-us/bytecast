AeroVista Revenue Engine Training Bundle

Files:
- index.html
- manifest.webmanifest
- service-worker.js
- assets/icon-192.png
- assets/icon-512.png
- assets/bytecast-ep3-revenue-spoken.mp3

What to do:
1. Replace assets/bytecast-ep3-revenue-spoken.mp3 with your final spoken ByteCast audio.
2. Host the folder on a local or web server. PWA install and service worker behavior require HTTP/HTTPS, not file://
3. Open index.html through the server.

Optional quick local server examples:
- Python: python -m http.server 8080
- Node: npx serve .
