# AeroVista Root SOT Pack

Included:
- `SOT.json` — AeroVista project root Source of Truth template
- `AEROVISTA_SOT_TEMPLATE_GUIDE.md` — field guide and recommended layout
- `EXAMPLE_NOTES.md` — quick notes on how to use it

Use:
1. Copy `SOT.json` to the root of an AeroVista project.
2. Update project metadata, stack, ports, and deploy path.
3. Add canon files and required paths.
4. Keep one SOT manifest per tracked project.
