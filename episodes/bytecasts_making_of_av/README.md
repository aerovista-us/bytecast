# ByteCasts Making of AV

This folder is the canonical home for ByteCasts Making of AV.

## Active entrypoints

- `index.html` - public landing page for the series.
- `bytecast.html` - current audio library/player experience.
- `audio/` - bundled episode audio files.
- `docs/` and `images/` - supporting source/reference material.

## Retired paths

The root-level `bytecasts_making_of_av/` path is retired and now exists only as a compatibility/reference notice. Do not add new feature work there.

## Legacy references

These files are retained here as reference material from earlier builds:

- `css_player.txt`
- `java.txt`
- `Now Playing bar HTML.txt`
- `echostory_bytecast_page.html`
- `pages/`
- `merch/`

If a feature from those older references should return, rebuild it into `index.html` or `bytecast.html` here instead of reviving the old root path.
