AV Apparel Contributor Onboarding — Full Training Outline
0) ENTRY: “Start Here” (3–5 minutes)
Purpose: set expectations + choose your lane + show how work moves from idea → live listing.

Includes

What AV Apparel is (brand + store + systems)

What “catalog hygiene” means

The 3 core tools (Printful / Square / Site catalog)

The golden rule: one source of truth + no drift

Interactive

“Pick your lane” selector (Maintainer / Copy & Naming / Pricing / Collections / QA)

60-second readiness check (quiz)

Slides (5–7)

The pipeline diagram

The “drift” problem + why it kills trust

What success looks like (curated store, consistent pricing, clean collections)

1) MODULE A: Welcome + Culture + How You Win Fast
✅ Use ByteCast #1 style (upbeat welcome for apparel helpers)

Learning outcomes

Understand how contributors fit in

Know the 3 lanes + access gating

Know what “first win” looks like in apparel

Interactive

“Choose your first win” menu

Micro-commitment: select one starter task and generate a checklist

Slides (8–10)

Lanes A/B/C (but apparel-focused)

What to do / what not to do

“First PR / first catalog update” path

2) MODULE B: Catalog Maintainer Bootcamp (Your Day-to-Day)
✅ Use ByteCast #3 (maintaining/organizing/updating/adding/removing/prices on site)

Learning outcomes

Safely add/remove products without breaking collections/pages

Keep titles, variants, images, tags consistent

Understand how to prevent drift across systems

Interactive

“Spot the drift” game: show 5 listings with hidden issues (wrong price, wrong image, orphaned collection, mismatched variants, duplicate product)

A guided “change request” form (they fill it, it outputs a work order)

Slides (12–15)

Catalog anatomy: Title / Images / Variants / Description / Tags / Collection / Status

Add workflow (checklist)

Remove workflow (checklist + orphan checks)

Update workflow (price, title, images, variants)

The “do not ship” list (broken thumbnails, mismatched sizes, inconsistent naming)

3) MODULE C: Pricing Discipline (Margin + Brand + Size Ladders)
Learning outcomes

Price ladders (tee vs hoodie vs premium)

Avoid loss-selling when base costs change

When to discount, when not to

How to audit pricing drift

Interactive

Pricing simulator: “Given base cost + target margin → suggested retail”

“Find the loss leaders” exercise: which SKUs are underwater?

Slides (10–12)

Retail price = cost + margin + brand positioning

Size upcharge policy

Discount rules + guardrails

Pricing audit checklist

4) MODULE D: Naming, Copy, and Collections (Make It Feel Curated)
✅ Use ByteCast #2 (welcome aimed at maintainers/organizers) as the opener tone here too

Learning outcomes

Title format standards

Copy format standards (hook → benefits → fit/care note)

Collections strategy (Core / Drops / Seasonal / Collabs)

Tagging standards (filtering + discovery)

Interactive

“Rewrite this listing” challenge (before/after)

“Build a collection” drag-and-drop (group products into a drop page)

Slides (10–12)

Title rules + examples

Copy framework + examples

Collection rules + examples

Tag taxonomy (simple, enforced)

5) MODULE E: QA + Release Ritual (Before It Goes Live)
Learning outcomes

Preflight checklist for every change

How to test the storefront like a customer

Mobile QA, broken image checks, variant checks, cart checks

Interactive

“QA scavenger hunt”: find 10 issues on a staging/demo catalog

Pass/fail gate: must score 80%+ to publish changes unsupervised

Slides (8–10)

The preflight list

Top 10 failure modes

Example of a perfect change request + PR notes

6) MODULE F: Access & Trust Ladder (How You Level Up)
Learning outcomes

What access is needed for what actions

How reviews work

How to earn publishing permissions

Interactive

“Trust Ladder” progress bar (auto-updates after quizzes + merged work)

Slides (6–8)

Roles: Viewer → Editor → Maintainer → Publisher

Review requirements per role

What gets escalated

Where Your 3 Existing ByteCasts Fit (clean placement)
ByteCast #1 (Welcome apparel helpers) → Module A opener

ByteCast #3 (Maintaining/organizing/updating/prices on site) → Module B core

ByteCast #2 (Welcome for maintainers/organizers/updaters) → Module D opener (or as a short “mid-course reset”)

Interactive System Buildout Ideas (for the full onboarding platform)
A) “Choose Your Role” onboarding wizard
User selects:

Maintainer (catalog operations)

Copy & naming

Pricing

Collections

QA

Wizard outputs:

recommended modules

required quizzes

first-win checklist

starter tasks

B) Staging Catalog “Practice Mode”
A fake catalog with intentional issues:

duplicate SKUs

mismatched images

wrong prices

broken collection links

out-of-stock items still live

They practice fixes → system grades them.

C) Work Order Generator
They fill:

change type (add/remove/update)

affected products

pricing change

collection changes

QA steps

It outputs a standardized ticket + PR template.

D) Drift Monitor Checklist (manual first, automated later)
Manual weekly audit:

pricing drift

orphaned products

duplicate titles

broken thumbnails

variant mismatch

Later: automate checks in CI (linting catalog JSON, image URL validation, duplicate detection).
